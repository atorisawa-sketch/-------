# ✍️ プロの添削くん

**「AIを、あなたの専属編集者に。」**
プロの編集者レベルの校閲・推敲を、あなたのブラウザから直接体験できるツールです。

---

### 🧐 これは何ができるのか？

このアプリは、単なる自動修正ツールではありません。

* **「プロの視点」でのリライト**: 誤字脱字の修正はもちろん、読みにくい文章を論理的で魅力的な文章へと書き換えます。
* **「納得」の解説機能**: ただ直すだけでなく、「なぜそのように直したのか」という編集意図をセットで提示します。
* **高精度なAIモデル**: 業界最先端の **Gemini 2.5 Flash** を搭載。ビジネスメール、SNS投稿、ブログの下書きを瞬時に磨き上げます。

### 🛡️ なぜ安心なのか？

「入力した文章や個人情報が、開発者に盗み見られるのでは？」という心配は不要です。

* **BYOK (Bring Your Own Key) 方式**: ユーザー自身の API キーを使用するため、AI との通信はあなたのブラウザと AI 間で直接行われます。
* **データの非保存**: 入力した文章や API キーは、開発者のサーバーやデータベースには一切保存されません。
* **高い透明性**: このアプリはオープンソース（ソースコードが公開された状態）であり、不正なデータ送信がないことを誰でも確認できます。

### 🚀 どうやって使うのか？

たったの 3 ステップで、プロの添削が始まります。

1. **API キーを準備**: [Google AI Studio](https://aistudio.google.com/) にログインし、無料で API キーを発行します。

   ※ Gemini 2.5 Flash は、1日あたり最大 500 リクエストまで無料で利用可能です。

2. **設定欄に入力**: アプリ画面上部の「Gemini API Key 設定」に、コピーしたキー（`AIza...` で始まるもの）を貼り付けます。

   ※ キーはあなたのブラウザに安全に保存され、外部（サーバー）に送信されることはありません。

3. **文章を送る**: 添削したいテキストを貼り付けて「添削する」ボタンを押すだけ。最新 AI による「プロの結果」が数秒で返ってきます。
